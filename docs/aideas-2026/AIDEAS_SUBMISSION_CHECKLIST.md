## 🆘 Troubleshooting

### Issue: AWS Deployment Fails

**Solutions:**
1. Check AWS credentials: `aws sts get-caller-identity`
2. Verify Bedrock access in your region
3. Check CDK bootstrap: `cdk bootstrap`
4. Review CloudFormation events in AWS Console
5. Check deployment script logs

### Issue: Article Won't Publish

**Solutions:**
1. Verify all required fields filled
2. Check image file sizes (under 2MB)
3. Ensure tags are correctly formatted
4. Try different browser
5. Contact AWS Builder Center support

### Issue: Video Won't Embed

**Solutions:**
1. Verify YouTube video is public/unlisted
2. Use YouTube embed code (not just URL)
3. Check video isn't age-restricted
4. Try re-uploading video
5. Use alternative video hosting if needed
